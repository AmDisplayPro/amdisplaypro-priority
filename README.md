## 🔧 What This Project Does

**AmDisplayPro** is a WordPress plugin that allows affiliates to display Amazon products on their websites with professional templates, automatic geolocation, and full support for Amazon's Creators API.

### Key Features (as of March 2026)

| Feature | Description |
|---------|-------------|
| **Creators API Support** | Full OAuth 2.0 authentication with token caching |
| **Regional Endpoints** | 2.1 (North America), 2.2 (Europe), 2.3 (Far East) |
| **OffersV2 Pricing** | Uses Amazon's latest pricing data format |
| **15 Templates** | Box, Compact, Simple, Space Saver, Spotlight, Comparison, List, and more |
| **Geolocation** | Automatically redirects visitors to their local Amazon store |
| **Smart Caching** | Configurable cache to respect API rate limits |
| **Multi-Region Tags** | Support for multiple Amazon associate tags per region |
